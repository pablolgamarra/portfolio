# Vive Telecom S.A

Ciudad del Este - Paraguay

Vive Telecom es un ISP que provee conexión de internet y servicio técnico tercerizado a locales del shopping Paris y shopping Bonita Kim, en el microcentro de Ciudad del Este.

## Puesto

Encargado de Soporte de T.I.

## Fecha de Inicio

Marzo - 2022

## Fecha de Fin

Diciembre - 2022

## Descripcion

Primera experiencia laboral profesional dentro del área de tecnología, como encargado de soporte brindé asistencia técnica a clientes dentro del shopping parís ya sea a nivel de hardware o software (N1). Además brindaba soporte a sistemas del shopping, como software para gestion de contenido (CMS) de videowalls, sistemas de acceso y gestión de estacionamiento y sistemas de cibercafé.

## Principales tecnologías con las que trabajo

- Mikrotik Routers & Switches
- Windows Server (2008)
- Linux Servers
- Sistemas de control de acceso a estacionamiento CAME/NEPOS
- Excel Macros (Excel VBA)
- CMS de diferentes vendor

## Principales tareas

- Gestion de activos de T.I.
- Compra de equipamientos electrónicos
- Mantenimiento de hardware para estaciones de trabajo
- Mantenimineto de hardware de servidores fisicos
