# Set-Signatures

Solución de automatización desarrollada en PowerShell para administrar e implementar firmas de correo electrónico de forma centralizada en organizaciones que utilizan Microsoft 365. El proyecto permite generar y asignar firmas corporativas estandarizadas a usuarios específicos, grupos de usuarios o a toda la organización.

## Objetivo

Garantizar la uniformidad de la identidad corporativa en las comunicaciones por correo electrónico, reduciendo la carga administrativa asociada a la creación y mantenimiento manual de firmas.

## Tecnologías

* PowerShell
* Microsoft 365
* Exchange Online
* Active Directory
* Microsoft Graph

## Funcionalidades principales

* Generación automática de firmas corporativas.
* Asignación de firmas a usuarios individuales o grupos específicos.
* Despliegue masivo para toda la organización.
* Integración con datos de Active Directory y Microsoft 365.
* Personalización dinámica utilizando atributos de usuario.
* Administración centralizada de plantillas.
* Actualización automática ante cambios en la información corporativa.

## Caso de uso

El sistema es utilizado para mantener una imagen corporativa consistente en todas las comunicaciones de correo electrónico. Los datos de cada usuario, como nombre, cargo, departamento, teléfono o ubicación, son obtenidos automáticamente desde el directorio corporativo y aplicados a plantillas predefinidas.

Esto permite que cualquier modificación en la estructura organizacional o en la identidad visual pueda ser desplegada de forma masiva sin intervención manual sobre cada estación de trabajo.

## Desafíos técnicos

* Integración con servicios de Microsoft 365 y Exchange Online.
* Obtención y procesamiento de atributos de usuario desde Active Directory.
* Generación dinámica de firmas personalizadas a partir de plantillas reutilizables.
* Gestión de diferentes diseños según áreas o grupos organizacionales.
* Automatización del despliegue en entornos con múltiples usuarios y sucursales.

## Resultado

La solución redujo significativamente el tiempo dedicado a la administración de firmas corporativas, eliminó inconsistencias entre usuarios y facilitó la aplicación de cambios de imagen institucional a nivel organizacional mediante un proceso completamente automatizado.
