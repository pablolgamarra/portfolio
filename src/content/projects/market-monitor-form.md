# Market Monitor Form

WebPart desarrollado sobre SharePoint Framework para el equipo de marketing, orientado al registro y seguimiento del comportamiento del mercado en relación a ventas de productos a clientes.

La solución permite centralizar información relevante para análisis comercial y apoyar la toma de decisiones basada en datos dentro del ecosistema de Microsoft 365.

## Objetivo

Facilitar al equipo de marketing una herramienta integrada dentro de SharePoint para registrar, visualizar y monitorear el avance del mercado en términos de ventas, evitando el uso de herramientas externas o dispersión de información.

## Tecnologías

* SharePoint Framework
* React
* TypeScript
* Microsoft

## Funcionalidades principales

* Registro de información de ventas y comportamiento del mercado.
* Visualización de datos para análisis del equipo de marketing.
* Integración directa con SharePoint Online.
* Interfaz basada en formularios para carga estructurada de datos.
* Acceso controlado mediante permisos de Microsoft 365.
* Persistencia de información en listas de SharePoint.

## Enfoque técnico

* Desarrollo de WebPart con SPFx integrado en SharePoint Online.
* Uso de React para la construcción de componentes reutilizables.
* Tipado estricto con TypeScript para mayor robustez del código.
* Consumo de listas de SharePoint como fuente de datos principal.
* Diseño orientado a la experiencia de usuario dentro del ecosistema Microsoft 365.

## Resultado

Se implementó una herramienta interna que centraliza la recolección de datos del mercado, permitiendo al equipo de marketing analizar tendencias de ventas de forma más estructurada y reducir la dependencia de reportes manuales o herramientas externas.
