# Kurundu - Sitio Web Catálogo

Plataforma web desarrollada para una empresa del rubro de productos artesanales, orientada a la exhibición de catálogo de productos y fortalecimiento de la presencia digital de la marca.

El sistema permite gestionar contenido de forma dinámica mediante un CMS y ofrecer una experiencia rápida y optimizada para los usuarios finales.

## Objetivo

Centralizar la presentación del catálogo de productos artesanales en una interfaz web moderna, fácil de mantener y escalable, sin depender de ediciones manuales del frontend.

## Tecnologías

* Astro
* Sanity Studio
* Vercel

## Arquitectura

* Frontend estático generado con Astro para máximo rendimiento.
* Contenido administrado mediante Sanity CMS (Sanity Studio).
* Despliegue automatizado en Vercel con integración continua.

## Funcionalidades principales

* Catálogo de productos artesanales administrado desde CMS.
* Renderizado estático con actualización dinámica de contenido.
* Diseño responsivo optimizado para mobile-first.
* Estructura SEO optimizada para indexación en buscadores.
* Separación clara entre contenido y presentación (headless CMS).

## Enfoque técnico

* Uso de arquitectura JAMstack para mejorar performance y escalabilidad.
* Integración de CMS headless para desacoplar contenido del frontend.
* Optimización de carga mediante generación estática en build time.
* Deploy automatizado con preview deployments en cada cambio.

## Resultado

Se obtuvo una plataforma liviana, escalable y fácil de mantener, que permite a la empresa actualizar su catálogo sin intervención técnica y mejora la visibilidad digital de sus productos artesanales.
